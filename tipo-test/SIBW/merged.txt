Tema 1: Introducción a los SIBW
Pregunta 1: ¿Cuál es la principal característica distintiva de un Sistema de
Información Basado en Web (SIBW) en comparación con otros sistemas de
información?
a) Siempre requiere un servidor de base de datos local.
b) Utiliza navegadores web como interfaz de usuario para acceder a la información y
funcionalidades.
c) Solo puede ser accedido desde dispositivos móviles.
d) Se desarrolla exclusivamente con lenguajes de programación de escritorio.
Respuesta Correcta: b) Utiliza navegadores web como interfaz de usuario para acceder
a la información y funcionalidades.

Pregunta 2: ¿Cuál de los siguientes protocolos es fundamental para la transferencia
de documentos hipermedia en la Web?
a) FTP (File Transfer Protocol)
b) SMTP (Simple Mail Transfer Protocol)
c) HTTP (Hypertext Transfer Protocol)
d) SSH (Secure Shell)
Respuesta Correcta: c) HTTP (Hypertext Transfer Protocol)

Pregunta 3: ¿Qué es el HTML (HyperText Markup Language) en el contexto de la
Web?
a) Un lenguaje de programación para crear interactividad en las páginas web.
b) Un protocolo para el intercambio seguro de información.
c) Un lenguaje de marcado utilizado para estructurar el contenido de las páginas web.
d) Un sistema de gestión de bases de datos para la Web.
Respuesta Correcta: c) Un lenguaje de marcado utilizado para estructurar el contenido
de las páginas web.

Pregunta 4: ¿Cuál de las siguientes afirmaciones describe mejor el concepto de
"hipertexto"?
a) Texto plano sin formato que se almacena en una base de datos.
b) Texto que contiene enlaces a otros textos o documentos.
c) Un tipo de texto que solo puede ser leído en dispositivos móviles.
d) Texto encriptado para garantizar la seguridad de la información.
Respuesta Correcta: b) Texto que contiene enlaces a otros textos o documentos.

Pregunta 5: ¿Qué papel juega un "servidor web" en la arquitectura de un SIBW?
a) Es el dispositivo del usuario final que solicita la información.
b) Almacena y procesa las peticiones de los clientes, enviando las páginas web
solicitadas.
c) Se encarga exclusivamente de la seguridad de la red.
d) Traduce el código HTML a un formato legible por el navegador.
Respuesta Correcta: b) Almacena y procesa las peticiones de los clientes, enviando las
páginas web solicitadas.

Pregunta 6: ¿Qué significa que HTTP es un protocolo "sin estado" (stateless)?
a) Que no puede transmitir información dinámica.
b) Que cada petición del cliente es independiente de las anteriores, sin que el servidor
guarde memoria de ellas.
c) Que no requiere una conexión a Internet constante.
d) Que los datos transferidos no tienen un formato definido.
Respuesta Correcta: b) Que cada petición del cliente es independiente de las
anteriores, sin que el servidor guarde memoria de ellas.

Pregunta 7: ¿Qué tecnología se utiliza principalmente para dar estilo y presentación

c) CSS (Cascading Style Sheets)
d) PHP

Respuesta Correcta: c) CSS (Cascading Style Sheets)

Pregunta 8: ¿Qué funcionalidad principal aporta JavaScript a una página web?
